May I Interest You in a Chuck Norris Joke in this Trying Quarantine?









The API I selected, Chucknorris.io, is an incredibly useful API that, when called, returns a Chuck Norris fact in JSON-object format, along with an ID for the fact and a URL through which it can also be accessed. These facts are completely factual and not at all absurdist jokes that riff off of pop culture and modern society. Each fact can be accessed using the URL contained in its associated JSON object, or completely at random. Random selection can be done either from the complete list of all Chuck Norris facts stored in the API, or from a certain subset of facts within a certain category (i.e. movie-related facts, sports-related facts)


In order to get my desired data from the Chuck Norris API, I utilized a GET request with the URL the API's website cites to get random Chuck Norris facts from a certain category. I specified the "dev" category within the URL, meaning that the API would return random Chuck Norris facts that related to computer science.


Because the API is very niche and is dedicated to a specific purpose, one would have to get creative to use this API in their own web applications. One potential application one could use this API for is a Chuck Norris biographical website, if the creator wanted to add humor and show Chuck Norris' influence and legacy on the Internet and beyond. Another potential application would be a niche and weirdly-developed semi-random number generator, where one would request a completely random Chuck Norris fact, find the length of the stringified JSON object, and then output the length as a semi-random number.


The link where I found the API is https://github.com/public-apis/public-apis#personality. JJ Stewart, Chuck Norris and out.
